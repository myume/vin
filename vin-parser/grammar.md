# Grammar

Pretty simple for now. Just a sequence of keyboard inputs

```grammar
PROGRAM ::= STATEMENT*

STATEMENT ::= (KEYBOARD_COMMAND) NEWLINE

REPEAT_LOOP ::= "REPEAT" NUMBER "{" NEWLINE STATEMENT* "}"

KEYBOARD_COMMAND ::= KEYPRESS_COMMAND | SENDKEY_COMMAND | HOLDKEY_COMMAND | RELEASEKEY_COMMAND

KEYPRESS_COMMAND ::= "PRESS" KEY

HOLDKEY_COMMAND ::= "HOLD" KEY

RELEASE_COMMAND ::= "RELEASE" KEY

SENDKEY_COMMAND ::= "SEND" STRING

KEY ::= \<keyboard key\>
```
